# 合成数ハンター (Composite Hunter)

素因数分解を楽しく学べる教育ゲームアプリです。

## ゲームについて

「合成数ハンター」は、素数を武器として合成数の敵と戦う戦略的バトルゲームです。プレイヤーは素因数分解を駆使して敵を倒し、数学的思考力を楽しみながら身に付けることができます。

## 特徴

- 📚 **教育的**: 素因数分解と素数の概念を実践的に学習
- 🎮 **ゲーム性**: 戦略的バトルシステムで楽しく学習
- 📈 **段階的学習**: レベルに応じた難易度調整
- 🏆 **実績システム**: 達成感を高める各種アチーブメント
